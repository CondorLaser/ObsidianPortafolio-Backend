# CLAUDE.md — ObsidianPortafolio-Backend

## Atribución de Claude: prohibida

NO agregues atribución de Claude a ningún artefacto del repositorio. Específicamente:

- **Commits**: no añadas `Co-Authored-By: Claude <noreply@anthropic.com>` (ni ninguna variante: `Claude Code`, `Claude Opus`, `Anthropic`, `claude.ai`, etc.) en el trailer ni en el cuerpo del mensaje. El autor y co-autor de cada commit es el usuario.
- **PRs (título, descripción, comentarios)**: no incluyas frases tipo `Generated with Claude Code`, links a `claude.com/claude-code`, emojis de robot, ni cualquier marca que indique que Claude participó.
- **Issues, releases, tags, CHANGELOG, README**: mismo criterio. No menciones a Claude/Anthropic como contribuidor.
- **Código y comentarios en archivos**: no firmes con `// generated by Claude` ni equivalentes.

Esto aplica a todas las operaciones git/gh que ejecutes en este repo, incluso si la plantilla por defecto de Claude Code sugiere el trailer. Sobrescribe la plantilla y omite la línea.

Si el usuario te pide explícitamente agregar atribución de Claude en un caso puntual, hazlo solo para ese caso y no lo asumas como regla.
